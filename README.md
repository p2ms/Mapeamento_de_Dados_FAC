# Projeto de Extensão: Mapeamento de Dados

O código foi desenvolvido com o auxilio de IA para criar um modelo apresentável para demonstrar uma breve pesquisa de campo acerca dos temas de pesquisa dos docentes do Departamento de Estatística, Matemática Aplicada e Computação (DEMAC). _(Projeto realizado por um mês e descontinuado)_
